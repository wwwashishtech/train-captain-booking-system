## 🌟 Overview
The Train Captain is a fully-featured railway ticket booking desktop application inspired by IRCTC.  
It simulates a complete booking experience — from login to ticket confirmation — with a modern GUI.

---

## 🚀 Key Highlights
- End-to-end booking workflow
- Clean Tkinter-based UI
- Real-time clock integration
- Smart PNR generation
- JSON-based data persistence

---

## ✨ Features

### 🔐 Authentication
- Secure login system
- User session handling

### 🔍 Train Search
- Search trains between stations
- Dynamic results display

### 🚆 Train Selection
- View available trains
- Select train & class

### 👥 Passenger Management
- Add up to 6 passengers
- Input validation

### 💳 Payment System
- Card, UPI, Net Banking, Wallet

### 🎫 PNR Generation
- Unique PNR generation
- Checksum validation
- Ticket confirmation display

### 📜 Booking History
- View and filter bookings

### ⚙️ Settings
- User preferences

### ⏰ Real-Time Clock
- Live time and date display
